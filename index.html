Preguntas frecuentes

Dudas comunes antes de implementar Qontrolab.

Estas respuestas ayudan a entender cómo puede adaptarse Qontrolab a la operación real de un laboratorio, sanatorio, clínica médica o centro de diagnóstico.

¿Qontrolab sirve solo para laboratorios clínicos?

No. Está pensado para laboratorios clínicos, sanatorios, clínicas médicas y centros de diagnóstico que necesitan ordenar pacientes, órdenes, resultados, pagos y reportes.

¿Necesito instalar algo en cada computadora?

No. Qontrolab funciona como plataforma web, por lo que puede utilizarse desde navegador en computadoras, tablets o celulares autorizados.

¿Puede manejar varias sucursales?

Sí. La operación puede organizarse por sede o sucursal para mantener datos separados y una visión general de toda la institución.

¿Se pueden generar resultados en PDF?

Sí. El sistema permite entregar resultados en formato profesional, con una presentación adecuada para pacientes, médicos e instituciones.

¿Puede conectarse con equipos de laboratorio?

Sí, mediante Synapse. Según el equipo, se puede evaluar conexión por ASTM, HL7 o archivos planos para reducir digitación manual.

¿Qué pasa con el soporte?

Qontrolab contempla acompañamiento y soporte para resolver dudas, revisar procesos y ayudar al equipo durante la implementación y uso del sistema.
